# pharmacy-reimbursement-analysis
Power BI dashboard analyzing pharmacy reimbursement data (2019-2024) – Total montant: 941M, 56K patients, trends, demographics, and forecasting.


## Project Overview
This Power BI dashboard provides a comprehensive analysis of pharmacy reimbursement data ("montant") from 2019–2024.

**Key Metrics:**
- Total Reimbursement: **941.10 Million**
- Unique Patients: **56K**
- Distinct Medicaments: **1,855**
- Hospitals: **36K**
- Doctors: **40K**
- Average Montant per Patient: **16.96K**

The dashboard includes:
- Executive Summary
- Patient Demographics & Spending (by age, gender, blood type)
- Medicament Analysis (top drugs, annual/monthly trends)
- Hospital Performance
- Spending Trends (annual, monthly, daily)
- Forecasting (using Prophet and built-in forecasting)
- Intelligent Q&A feature

## Key Insights
- Top 5 medicaments (Aspirin, Penicillin, Ibuprofen, Lipitor, Paracetamol) account for ~99% of costs
- Highest spending in 60+ age group (38%)
- Near-equal gender distribution
- Clear monthly peaks (June, July, October) and daily drop-off at month-end
- Stable annual growth with 2024 partial data

## Recommendations from Analysis
- Negotiate pricing for top 5 medicaments
- Audit high-spending hospitals
- Target health programs for 40-60+ age groups
- Improve cashflow planning around month-end patterns
